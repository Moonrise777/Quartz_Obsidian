- Un sistema es el conjunto de componentes que interactúan entre sí para lograr un objetivo común.

- Son los programas de computadora, documentos asociados y la configuración de los datos que se necesitan para hacer que estos programas operen de manera correcta.

- Los problemas que se originan al desarrollar los sistemas se conocen como crisis del software.

- Se le denomina  a la aplicación de los conocimientos de software para la construcción de programas.

- Es el software como servicio el que provee generalmente a través de infraestructura externa de T.I  en la nube

- Un ejemplo de  puede ser el e-siima

- El software como producto es el que instalamos en nuestros dispositivos o que compramos en caja.

- La IA es un ejemplo de Software Inteligente de las categorías del software
